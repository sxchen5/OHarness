# 贪吃蛇网页游戏

经典贪吃蛇浏览器游戏，纯 HTML/CSS/JavaScript 实现。

## 快速开始

1. 用浏览器直接打开 `index.html`，或
2. 启动本地静态服务器：

```bash
npx serve .
```

然后访问提示的地址（通常为 http://localhost:3000）。

## 操作说明

| 按键 | 功能 |
|------|------|
| ↑ ↓ ← → | 控制蛇的移动方向 |
| 空格 | 暂停 / 继续 |
| 重新开始按钮 | 重置游戏 |

## 规则

- 吃到红色食物得 10 分，蛇身变长
- 撞墙或撞到自己游戏结束
- 每吃 5 个食物速度加快（初始约 280ms/步，最快约 100ms/步）
- 最高分保存在浏览器 localStorage

## 开发与测试

```bash
npm test
```

游戏逻辑在 `game-logic.js`，可在 Node 环境单元测试；`game.js` 负责 Canvas 渲染与输入。
